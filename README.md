# Projeto

Este projeto consiste em um desafio da plataforma FrontendMentor, onde o objetivo deste desafio foi implementar um App funcional de Time Tracking usando os dados fornecidos pelo desafio.

<img width="1910" height="907" alt="frontendmentor-challenge-time-tracking" src="https://github.com/user-attachments/assets/c8c75045-5fdd-41a2-b7a8-0a6a8008a495" />

# Tecnologias usadas

- Tailwindcss para estilização
- NextJS com typescript

# Rodar o projeto

- Baixe ou clone o repositório
- Em seguida, instale as dependências
- Por fim, rode o comando npm run dev
